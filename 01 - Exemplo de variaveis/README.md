# 01 - Exemplo de Variáveis

## Objetivo

Esta seção apresenta os conceitos básicos de variáveis em Java.

Variáveis são utilizadas para armazenar dados na memória durante a execução do programa, permitindo que esses valores sejam utilizados, modificados e consultados ao longo do código.

---

## Exemplo Básico

```java
String nome = "João";
int idade = 25;

System.out.println(nome);
System.out.println(idade);
```

---

## Observações

- Toda variável possui um tipo.
- O tipo da variável determina quais valores ela pode armazenar.
- Em Java, variáveis devem ser declaradas antes de serem utilizadas.
- Variáveis locais devem ser inicializadas antes de serem lidas.
- O nome das variáveis normalmente segue o padrão *camelCase* (A primeira letra da primeira palavra é minúscula e as iniciais das palavras seguintes são maiúsculas, sem espaços ou hifens).

Exemplo:

```java
int idadeUsuario;
double precoProduto;
String nomeCompleto;
```
