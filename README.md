# Revisando Java

Repositório dedicado ao estudo, revisão e consolidação da linguagem Java através de anotações, exercícios, exemplos práticos e projetos desenvolvidos ao longo do processo de aprendizagem.

O objetivo deste repositório é registrar a evolução dos estudos, criar uma base de consulta para revisões futuras e servir como material de apoio para quem deseja aprender Java desde os fundamentos até tópicos mais avançados.

---

# O que é estudado neste repositório?

Ao longo deste repositório são abordados conceitos fundamentais da linguagem Java, incluindo:

## Fundamentos da Linguagem

- Identificadores Válidos
- Variáveis
- Tipos Primitivos
- Promoções Automáticas de Tipos
- Inferência de Tipo de Variável Local (`var`)
- Operadores
- Estruturas Condicionais
- Estruturas de Repetição (Loops)

Esses tópicos formam a base da programação em Java e são essenciais para a construção de qualquer aplicação.

---

## Programação Orientada a Objetos (POO)

- Classe `Object`
- Construtores
- Packages
- Modificadores de Acesso
- Métodos
- Getters e Setters
- Passagem por Valor e por Referência
- Membros `static`
- Métodos `static`
- Herança
- Polimorfismo
- Interfaces
- Classes Abstratas
- Imutabilidade

A Programação Orientada a Objetos é um dos pilares do Java e permite desenvolver sistemas mais organizados, reutilizáveis e fáceis de manter.

---

## Estruturas de Dados e Tipagem

- Arrays
- Generics
- Collections Framework

Esses recursos permitem armazenar, organizar e manipular dados de forma eficiente e segura.

---

## Recursos Modernos da Linguagem

- Annotations
- Expressões Lambda
- Interfaces Funcionais
- Programação Funcional

Esses recursos tornam o código mais moderno, expressivo e alinhado às práticas utilizadas nas versões atuais da linguagem.

---

## Tratamento de Erros

- Exceptions
- Tratamento de Exceções
- Captura e Propagação de Erros

Esses conceitos permitem criar aplicações mais robustas e resilientes.

---

## Entrada, Saída e Persistência de Dados

- Input e Output (I/O)
- Manipulação de Arquivos
- Serialização de Objetos (`Serializable`)

Esses recursos permitem salvar, recuperar e transferir dados entre aplicações e sistemas.

---

## Desenvolvimento de Interfaces Gráficas

- JavaFX
- CSS no JavaFX

Utilizados para criar aplicações desktop com interfaces gráficas modernas e personalizáveis.

---

## Exercícios

Cada seção possui exercícios práticos desenvolvidos para reforçar os conceitos estudados.
