# Exercício 1 - Soma de Valores Acumulados (`for`)

## Cenário

Você precisa criar um programa que calcule automaticamente a soma dos números de 1 até 10.

Ao invés de realizar cada soma manualmente, utilize uma estrutura de repetição para repetir o processo de forma automática.

---

## Requisitos

1. Crie uma variável inteira chamada `soma` e inicialize-a com `0`.
2. Utilize um laço para percorrer os números de `1` até `10`.
3. A cada repetição, some o valor atual à variável `soma`.
4. Após o término do laço, exiba o resultado obtido.

---

## Resultado Esperado

O programa deve calcular corretamente a soma de todos os números entre `1` e `10` e exibir o resultado no console.

---

## O que este exercício reforça?

* Estrutura `for`
* Variáveis acumuladoras
* Incremento de variáveis
* Controle de iterações
* Operadores de atribuição

# Exercício 2 - Contagem de Doces (`while`)

## Cenário

Uma mãe permitiu que seu filho coma no máximo três doces.

Seu programa deverá controlar essa contagem utilizando um laço `while`.

---

## Requisitos

1. Declare uma variável inteira chamada `contadorDoces`.
2. Inicialize-a com o valor `1`.
3. Utilize um laço `while` para repetir o processo enquanto o contador for menor ou igual a `3`.
4. Exiba a quantidade de doces consumidos a cada repetição.
5. Ao final, exiba uma mensagem informando que não é permitido comer mais doces.

---

## Resultado Esperado

O programa deve exibir a contagem dos doces consumidos e, ao final, informar que o limite foi atingido.

---

## O que este exercício reforça?

* Estrutura `while`
* Avaliação de condições
* Contadores
* Incremento de variáveis
* Repetição baseada em condição

---

# Exercício 3 - Soma de Números Inteiros Positivos (`do-while`)

## Cenário

Você deverá criar um programa que calcule a soma de todos os números inteiros positivos até um valor determinado.

Para isso, utilize um laço `do-while`.

---

## Requisitos

1. Declare uma variável inteira chamada `numero` e atribua um valor positivo.
2. Crie uma variável acumuladora chamada `soma`.
3. Utilize um laço `do-while` para realizar a soma dos valores.
4. Decremente o valor de `numero` a cada repetição.
5. Ao final, exiba o resultado da soma.

---

## Resultado Esperado

O programa deve calcular corretamente a soma de todos os números inteiros positivos até o valor informado.

---

## O que este exercício reforça?

* Estrutura `do-while`
* Diferença entre `while` e `do-while`
* Variáveis acumuladoras
* Decremento de variáveis
* Controle de repetições

---

## Desafio de Observação

Durante a implementação, tente responder:

* Quando utilizar um `for` em vez de um `while`?
* Em quais situações o `do-while` é mais adequado?
* Qual a principal diferença entre `while` e `do-while`?
* O que acontece se a condição de um `do-while` for falsa logo na primeira verificação?
* Por que a variável `soma` precisa ser inicializada antes do laço?

Não é necessário pesquisar imediatamente as respostas. Tente observar o comportamento do programa e tirar suas próprias conclusões antes de avançar.
