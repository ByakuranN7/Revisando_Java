public class Saudacao {
    public static void main(String[] args) {
        int num1 = 79;
        int num2 = 108;
        int num3 = 225;

        char letra1 = (char) num1; // 'O'
        char letra2 = (char) num2; // 'l'
        char letra3 = (char) num3; // 'á'

        System.out.println("Saudação: " + letra1 + letra2 + letra3); //Saída: "Saudação: Olá"

    }
}


Explicação do código:
Neste código, temos três números inteiros: num1, num2 e num3. Eles são convertidos para caracteres char usando (char).

num1 é 79, que corresponde ao caractere ‘O’.

num2 é 108, que corresponde ao caractere ‘l’.

num3 é 225, que corresponde ao caractere ‘á’.

Como resultado temos "Olá" impresso na tela. Isso é possível porque todo caractere tem um número correspondente na tabela Unicode (tabela de caracteres).
